HTML : HyperText Markup Language
-> Ngôn ngữ đánh dấu siêu văn bản
CSS  : Cascading Style Sheets
-> Ngôn ngữ định dạng web (tô màu,in đậm,in nghiêng ...)
JS : JavaScript
-> Các thao tác văn bản : Ẩn, hiện khi click, thay đổi nội dung hiển thị ...
